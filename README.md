# git-co-commit

Git co-commit is the npm package that enables multiple users to pair and commit at the same time.

Install it globally,

```bash
npm i -g git-co-commit
```

Use it in global,

```bash
commit
```

Install it locally,

```bash
npm i git-co-commit
```

For use it in local specify following in scripts of package.json

```js
"commit": "commit"
```

For run it

```bash
npm run commit
```
